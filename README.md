# 🧠 AI Health Insurance Predictor

🚀 An interactive Machine Learning web application that predicts healthcare insurance costs based on patient details using advanced models and data analysis.

---

## 📌 Project Overview

The **AI Health Insurance Predictor** is a data-driven application built using **Streamlit** that helps estimate insurance claim costs using user inputs like age, BMI, smoking habits, and medical conditions.

It combines **Machine Learning + Data Visualization + UI Design** to deliver a complete analytics dashboard experience.

## ✨ Features

- 🔮 Real-time Insurance Cost Prediction  
- 🤖 Multiple ML Models (Random Forest & Linear Regression)  
- 📊 Interactive Data Visualization Dashboard  
- 📌 Correlation Analysis (Feature Relationships)  
- 🎯 User-friendly modern UI (SaaS-style design)  

---

## 🖥️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **Libraries:**  
  - Pandas  
  - NumPy  
  - Matplotlib  
  - Seaborn  
  - Scikit-learn  

---

## 📊 Machine Learning Models

- 🌲 Random Forest Regressor  
- 📈 Linear Regression  

Both models are used to predict insurance cost, and an **average prediction** is also displayed for better estimation.

---

## 📂 Dataset Features

- Age  
- Gender  
- BMI (Body Mass Index)  
- Blood Pressure  
- Diabetic  
- Children  
- Smoker  
- Region  
- Claim Amount  

---

## 🔍 Key Insights

- 💰 Smoking has the highest impact on insurance cost  
- 📈 Higher BMI increases claim amount  
- 🎂 Age gradually affects insurance pricing  
- 🧬 Medical conditions increase risk factor
